# Logo Instructions

## Add Your Logo Here

Place your **logo.png** file in this directory: `wwwroot/images/logo.png`

### Logo Specifications:
- **File name**: logo.png
- **Recommended size**: 512x512 pixels or similar square dimensions
- **Format**: PNG with transparent background
- **Style**: Should work well on both white and gradient purple backgrounds

### Where the logo appears:
1. **Login Page** - White background with rounded corners
2. **Dashboard Sidebar** - Purple gradient background

### Note:
The logo will be automatically resized to fit the designated areas. The system uses `object-fit: contain` to maintain aspect ratio.
